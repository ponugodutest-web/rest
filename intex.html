YouTube API v3 Publishing Application

This application uses the YouTube Data API v3 to allow users to securely upload, publish, and manage videos on their own YouTube channels.

Purpose of the App

The purpose of this app is to automate YouTube video publishing using Google OAuth 2.0 authentication. The app only works after a user explicitly grants permission.

How It Works

  • User signs in using Google OAuth 2.0
  • User grants YouTube permissions
  • The app uploads or manages videos on the user's behalf

Data Access

The app only accesses YouTube data required for uploading and managing videos. No unrelated personal data is accessed.
