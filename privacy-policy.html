Privacy Policy

This Privacy Policy explains how this application handles user data when accessing the YouTube Data API v3.

Information Collected

The app collects only the information necessary to interact with YouTube, including OAuth access tokens and basic YouTube account identifiers.

Use of Information

Collected data is used exclusively to:

  • Upload videos to YouTube
  • Manage video metadata
  • Perform actions explicitly requested by the user

Data Storage

Authentication tokens may be stored securely to maintain user sessions. No data is sold or shared with third parties.

Third-Party Services

This app uses Google APIs and follows Google’s Privacy Policy.

User Control

Users can revoke access at any time via their Google Account security settings.
