Privacy Policy

This Privacy Policy describes how this application uses and protects user data when accessing the YouTube Data API v3.

Information We Collect

We only collect information that is necessary to interact with the YouTube API, such as authentication tokens and basic account data.

Use of Information

Collected data is used solely to provide YouTube-related functionality such as uploading, managing, and publishing videos.

Data Storage

OAuth tokens may be stored securely and are never shared with third parties.

Third-Party Services

This application uses Google APIs and complies with Google Privacy Policy.

User Control

Users may revoke access at any time via their Google Account security settings.

Changes

This policy may be updated from time to time. Any changes will be posted here.
